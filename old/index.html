About PayPals

A Splitwise-inspired learning project that demonstrates a full-stack approach: Spring Boot backend + vanilla frontend. The single-page demo is framework-free and built for quick demos.

  • Backend: Java, Spring Boot, JPA/Hibernate, MySQL
  • Frontend: HTML, CSS (cool neon theme), JavaScript
  • Build: Maven/Gradle — Deployable to Heroku / Docker / AWS

Developer Notes

Buttons are placeholders. Replace modal handlers with real fetch calls to your Spring Boot endpoints like GET /api/groups and POST /api/expenses.
