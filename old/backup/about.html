About PayPals

PayPals is a Splitwise-inspired project built as a learning & portfolio application.

Project Purpose

To demonstrate a full-stack approach: backend implemented with Java Spring Boot (REST APIs, JPA/Hibernate, MySQL) and frontend using only HTML, CSS & vanilla JavaScript for a polished demo UI.

Key Technologies

  • Java (JDK 17+)
  • Spring Boot (REST, DI)
  • MySQL
  • Hibernate / JPA
  • HTML / CSS / JavaScript (this frontend)

Developer Notes

This frontend is intentionally framework-free to show core web skills and integration points. Backend endpoints should map to the dummy actions shown on the UI.
